# 🔬📊 Raman AnalysiS for Petrus _(RASP)_

A Python-based toolkit for the analysis and visualization of Raman spectroscopy and mapping data.

---

## 🔍 Overview

**RASP** is a modular set of tools designed to support the processing, analysis, and visualization of raw Raman data — including both point spectra and spatially resolved Raman mapping.

This project aims to assist researchers, students, and developers in working with Raman data by combining flexibility, automation, and clarity in data exploration.

---

## 🎯 Objectives

- Provide reusable and modular code for Raman data analysis
- Enable preprocessing (e.g. smoothing, baseline correction, normalization)
- Support multiple plotting styles for spectra and maps
- Automate common steps for quick insights
- Facilitate future integration with microscopy or imaging tools

---

## 🛠️ Technologies & Libraries

- Python 3.x
- NumPy / Pandas – data manipulation
- Matplotlib / Seaborn / Plotly – visualization
- SciPy – signal processing
- scikit-learn – dimensionality reduction (e.g. PCA)
- OpenCV (optional) – image operations
- Jupyter – interactive analysis

---

## 🚀 Features

- ✅ Import Raman data from raw formats (e.g. CSV, TXT)
- ✅ Baseline correction, smoothing and normalization
- ✅ Spectral plotting and overlay
- ✅ Raman maps (2D spatial visualization)
- ✅ PCA and clustering (in progress)
- ✅ Modular structure for reuse in other projects
